# Banned words and phrases

Never use any of these unless the user explicitly used them in their input. Replace with simpler, concrete alternatives.

## Banned verbs

delve, leverage, utilize, harness, streamline, underscore, foster, showcase, accentuate, empower, elevate, optimize, reimagine, revolutionize, spearhead, synergize, navigate (metaphorical), embark, endeavor, illuminate, elucidate, unpack, ascertain

**Quick replacements:**
- delve → look at, examine, dig into
- leverage → use
- utilize → use
- harness → use, apply
- streamline → simplify, speed up
- underscore → show, highlight
- foster → encourage, build
- showcase → show, display
- empower → let, enable, give [someone] the ability to
- elevate → raise, improve
- optimize → improve, tune
- navigate (metaphor) → deal with, work through, handle

## Banned adjectives

pivotal, robust, innovative, seamless, cutting-edge, groundbreaking, transformative, vibrant, meticulous, multifaceted, holistic, unprecedented, state-of-the-art, bespoke, paramount, invaluable, profound, compelling, commendable

**Quick replacements:**
- pivotal → important, key
- robust → strong, solid
- innovative → new, original
- seamless → smooth
- cutting-edge → new, latest
- groundbreaking → new, first
- transformative → changed, changed how
- paramount → most important
- compelling → strong, convincing

## Banned nouns

tapestry, landscape (figurative), realm, synergy, paradigm, testament, underpinnings, interplay, cornerstone, bedrock, nexus, crucible, catalyst, beacon, treasure trove, game-changer, journey (figurative)

**Quick replacements:**
- landscape → field, area, market
- realm → area, field
- synergy → combined effect, cooperation
- paradigm → model, approach
- testament → proof, sign
- cornerstone → foundation, base
- catalyst → cause, trigger
- journey → process, experience, path

## Banned adverbs

moreover, furthermore, consequently, notably, importantly, certainly, indeed, additionally, specifically, respectively, arguably, fundamentally, inherently

These are filler. Cut them and the sentence almost always improves. If the logical connection between sentences isn't clear without a transition word, restructure instead of papering over it.

## Banned opener phrases

- "In today's [fast-paced/ever-evolving/competitive] [world/landscape]..."
- "In the rapidly changing world of..."
- "As we navigate the complexities of..."
- "In an era where..."
- "When it comes to..."
- "In the world of..."

Just start with the actual subject.

## Banned filler phrases

- "It's worth noting..."
- "It's important to note..."
- "It's no secret..."
- "The reality is..."
- "Here's the thing:"
- "Let's be honest:"
- "The truth is..."
- "At the end of the day..."
- "Let that sink in."

Cut these entirely. Say the thing directly.

## Banned false-drama contrasts

- "It's not [just] X, it's Y"
- "X isn't just Y; it's Z"
- "Not just X, but Y"
- "More than just X — it's Y"

Max once per piece, if at all. Usually you can just state Y without the theatrical setup.

## Banned rhetorical setups

- "This is where X comes in"
- "Enter: [thing]"
- "The question isn't X. It's Y"
- "And that's exactly why..."
- "But here's the kicker:"
- "The best part? [thing]"
- "Why does this matter? Because..."
- "So, what does this mean?"
- "Think about it."
- "Imagine [scenario]."
- "Picture this:"

## Banned closers

- "In conclusion/summary/essence..."
- "Overall,..."
- "All in all,..."
- "At its core,..."
- "The bottom line is..."
- "Moving forward,..."
- "Looking ahead,..."
- "As we move into [year]..."

Just end. The last paragraph should carry its own weight without announcing itself.

## Banned sentence openers

Never begin a sentence with any of these words:

Certainly, Moreover, Additionally, Furthermore, Consequently, Notably, Importantly, Specifically, Interestingly, Surprisingly, Remarkably, Undoubtedly, Essentially, Ultimately, Admittedly, Indeed

Start with the actual point instead.
